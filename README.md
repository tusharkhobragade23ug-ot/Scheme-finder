# Maharashtra Government Scheme Finder

# Project Overview
This project is a web-based application that helps users find government schemes based on their eligibility.  
Users can filter schemes using parameters such as income, caste, occupation, gender, disability status, ration category, and scheme category.

The system then displays relevant schemes that match the selected criteria.

# Live Demo
https://tusharkhobragade23ug-ot.github.io/Scheme-finder/

# Features

- Filter schemes based on:
  - Income
  - Caste
  - Occupation
  - Gender
  - Disability
  - Ration Category (BPL, APL, AAY, PHH)
  - Category (Education, Farmer, Women, Housing, etc.)

- Dynamic results display
- Reset filters option
- Feedback system (stored using localStorage)
- Clean and responsive UI using Bootstrap

# Technologies Used

- HTML
- CSS (Bootstrap)
- JavaScript
- JSON (for storing scheme data)

# Data Handling

Currently, the project uses a static "scheme.json" file to store scheme data.  
This approach is suitable for a small dataset and demonstration purposes.


# Limitations

- Data is static (not real-time)
- Feedback is stored locally (not shared across users)
- No backend/database integration

# Future Scope

- Integration with database (Firebase / MongoDB)
- Admin panel to update schemes dynamically
- User authentication system
- Real-time data updates
- Advanced recommendation system
